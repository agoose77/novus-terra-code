รายละเอียดแบบอักษร
================
      โดย นายไพศาล เตชจารุวงศ์ (mailto: phaisarn_te@hotmail.com)
      วันที่ 9 กรกฎาคม 2541 (Documented: 9 September 1998)

แบบอักษรทุกตัวในสารบบนี้เป็นแบบอักษรที่มีขนาดของอักขระเท่ากันทุกตัว(Fixed
Font/ Monospace Font) จุดประสงค์เพื่อใช้ในงาน Thai Programing และใน
Internet Web Browser( Netscape, Internet Explore) ซึ่งจะใช้ต่างกัน
ตามแบบอักษรดังนี้

1) Phaisarn98 (Paisan98.ttf)
      Phaisarn98 เป็นแบบอักษรที่ตรงตามมาตรฐาน Thai TrueType Font ที่ทาง
Microsoft แจกมาพร้อมกับ Windows95 เช่น แบบอักษรในตระกูล UPC ไม่ว่าจะ
ในด้าน Encoding: TIS620-2533 & UNICODE 2.0, Thai Script และ Point
Size ที่มีขนาดใกล้เคียงมาตรฐาน โดยได้กำหนดให้เป็นประเภท Fixed Font

จุดประสงค์การใช้งาน:
ก) Internet Web Browser บน Windows95 Thai/NonThai ที่ต้องใช้
Encoding Font เป็น Thai และประเภท Fixed-width Font
ข) โปรแกรมแก้ไขเอกสารทั่วๆไป เช่นใช้ในโปรแกรม Wordpad เป็นต้น
(ยกเว้นใน Microsoft Word 95/97 Thai ซึ่งให้ใช้ phaisarn.ttf แทน)

2) Phaisarn97 (Paisan97.ttf)
      แบบอักษร Phaisarn97 นี้มีจุดประสงค์การใช้งานและคุณสมบัติเช่นเดียวกับแบบ
อักษร Phaisarn98 ยกเว้นเรื่องขนาด points จะมีขนาดใหญ่กว่าตามมาตรฐานจริง
แต่หากเห็นว่าการแสดงผลของ Phaisarn98 ใน IE4 (Internet Explorer 4)
มีขนาดเล็กไปก็ใช้ชุดนี้แทน

3) Phaisarn (phaisarn.ttf)
      เนื่องจากโปรแกรม Microsoft Word 95/97 Thai มีการตรวจสอบผิดพลาด
ในการแสดงแบบอักษรประเภท Fixed Font จึงทำให้ไม่สามารถแสดงผลแบบอักษร
Phaisarn98 และ Phaisarn97 ได้อย่างถูกต้องได้ (สามารถทดสอบได้จากการ
พิมพ์รหัสไทยด้วยแป้นพิมพ์ไทยโดยใช้แบบอักษร Courier ซึ่งเป็นมาตรฐาน Fixed
Font ในภาษาอังกฤษจะพบว่าระยะของตำแหน่งแต่ละอักษรจะมีการเลื่อนเย้ไปมาซึ่ง
เมื่อเปลี่ยนเป็น Thai Fixed Font ก็ต้องมีการเลื่อนเย้ไปมาด้วยเช่นกัน) Font
ชุดนี้จึงใช้แทน Phaisarn98/97 ในโปรแกรม MS-Word95/97 Thai

จุดประสงค์การใช้งาน:
ก) Internet Web Browser บน Windows95 Thai/NonThai ที่มี
Encoding Font เป็น Thai ประเภท Proportional Font
ข) ใช้ในโปรแกรมแก้ไขเอกสารทั่วๆไปรวมทั้ง MS Word 95/97 Thai

4) PhaisarnFixed (PaisanFX.ttf)
      เป็นแบบอักษรตามมาตรฐาน ASCII/ISO8859-1 หรือ Western Script

จุดประสงค์การใช้งาน:
ก) Internet Web Browser บน Windows95 Thai/NonThai ที่ไม่มี
Encoding Font เป็น Thai ประเภท Fixed-width Font
ข) โปรแกรมแก้ไขเอกสารทั่วๆไป (ยกเว้น Microsoft Word 95/97 Thai)
โดยเฉพาะที่ระบบที่ไม่ใช่ Windows95 Thai

5) PhaisarnMono (PaisanMN.ttf)
      แบบอักษร PhaisarnMono นี้มีจุดประสงค์การใช้งานและคุณสมบัติเช่นเดียวกับแบบ
อักษร PhaisarnFixed ยกเว้นเรื่องสระและวรรณยุกต์จะมีขนาดความกว้างด้วย

จุดประสงค์การใช้งาน:
ก) telnet บนระบบ Windows95
ข) Netscape Mail บนระบบ UNIX/Linux

หมายเหตุ
- การแสดงผลบน telnet ของ Windows95 อยู่ระหว่างการปรับปรุง
